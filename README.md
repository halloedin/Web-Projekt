Die Idee hinter dieser Web Anwendung ist eine übersichtliche und simple ToDo Liste mit mehreren Aufgaben erstellen zu können um den Alltag besser organisieren zu können. Hierbei gibt es hilfreiche Funktionen wie z.B. den Check oder die Mülltonne um einen Überblick über mehrere Aufgaben zu haben. Diese sagen aus in welchem Prozess sich die Aufgabe befindet.
